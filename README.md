# Question Number Generator

一個簡潔、視覺風格強烈的網頁工具，用於課堂、競賽或活動中隨機將題目分配給各組別。

## 功能特色

- 自訂總組數，彈性調整參與人數
- 可排除特定組別（例如缺席或棄權的組）
- 輸入任意數量的題目，每行一題
- 一鍵隨機打亂並分配題目給各組
- 滑順的動畫顯示結果，介面清晰易讀

## 使用方式

1. 開啟 `index.html`（直接用瀏覽器即可，無需安裝任何套件）
2. 設定總組數
3. 點選要排除的組別（已排除的組會顯示為刪除線）
4. 在文字區塊中輸入題目，每行一題
5. 按下 **GENERATE** 按鈕，系統會隨機將題目分配給各參與組別

## 技術說明

純前端實作，不依賴任何外部函式庫或框架。

- HTML / CSS / Vanilla JavaScript
- 使用 Fisher-Yates 演算法隨機打亂題目順序
- 響應式設計，支援行動裝置瀏覽
